This project is an interactive Power BI dashboard designed to analyze and visualize Blinkit sales performance. The dashboard provides clear insights into sales trends, product performance, customer behavior, and revenue distribution using dynamic visuals and filters.

1. Key Features
Total sales, orders, and revenue overview.
Category-wise and product-wise sales analysis.
City/region-wise performance insights.
Monthly and yearly sales trends.
Interactive slicers for better data exploration.
Clean and user-friendly dashboard design.

2. Tools & Technologies
Power BI
Data Cleaning & Transformation
Data Visualization
DAX (Measures & Calculations)

3. Objective
To help stakeholders quickly understand business performance, identify top-performing products and categories, and make data-driven decisions through visual analytics.
